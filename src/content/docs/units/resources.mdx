## Libros

- [The Elements of Computing Systems: Building a Modern Computer from First Principles](https://mitpress.mit.edu/books/elements-computing-systems-second-edition). Este libro es un recorrido por el funcionamiento de un sistema de cómputo desde las compuertas que componen el computador hasta un videojuego conocido como tetris. Este texto soporta un curso muy popular llamado [from nand to tetris](https://www.nand2tetris.org/).
- [Computer Systems: A Programmer's Perspective](https://www.pearson.com/us/higher-education/program/Bryant-Computer-Systems-A-Programmer-s-Perspective-3rd-Edition/PGM2476825.html). Una verdadera joya que aborda la arquitectura y organización de los sistemas de cómputo desde la perspectiva del programador.
- [Operating Systems: Three Easy Pieces](https://pages.cs.wisc.edu/~remzi/OSTEP/). Este excelente libro gratuito de sistemas operativos hace un recorrido por las abstracciones más importantes que este software ofrece.
- [Dive into Systems](https://diveintosystems.org/). Este texto gratuito captura la esencia de nuestro curso. El material que está allí te permitirá complementar y profundizar todo lo que vas a estudiar.
- [Web Browser Engineering](https://browser.engineering/).
